# =====================================================
# GreekBudget2025
# Εφαρμογή Διαχείρισης Κρατικού Προϋπολογισμού
# =====================================================

Η GreekBudget2025 είναι μια εφαρμογή οικονομικής διαχείρισης (Budget Management Application),
αναπτυγμένη σε Java (JavaFX). Η εφαρμογή επιτρέπει την καταγραφή, ανάλυση και σύγκριση
οικονομικών δεδομένων (έσοδα, έξοδα, κρατικές δαπάνες) με στόχο την καλύτερη κατανόηση
και διαχείριση του ετήσιου προϋπολογισμού.

Το project έχει υλοποιηθεί στο πλαίσιο ακαδημαϊκού / εκπαιδευτικού σκοπού και ακολουθεί
βασικές αρχές καθαρού κώδικα, διαχωρισμού λογικής και διεπαφής χρήστη, καθώς και χρήσης
τοπικής βάσης δεδομένων.

# -----------------------------------------------------
# 1. ΟΔΗΓΙΕΣ ΜΕΤΑΓΛΩΤΤΙΣΗΣ (BUILD)
# -----------------------------------------------------

# Απαιτήσεις
# - Java JDK 17 ή νεότερη
# - Apache Maven

# Μεταγλώττιση του project
mvn clean install

# Η εντολή:
# - κατεβάζει τα dependencies
# - μεταγλωττίζει τον κώδικα
# - δημιουργεί τον φάκελο target/

# -----------------------------------------------------
# 2. ΟΔΗΓΙΕΣ ΕΚΤΕΛΕΣΗΣ (RUN)
# -----------------------------------------------------

# Εκτέλεση μέσω Maven (JavaFX)
mvn javafx:run

# Εκτέλεση JAR με JavaFX modules
java --module-path $PATH_TO_FX --add-modules javafx.controls,javafx.fxml,javafx.graphics -jar greekbudget2025.jar
