# GreekBudget2025
## Εφαρμογή Διαχείρισης Κρατικού Προϋπολογισμού

Η GreekBudget2025 είναι μια εφαρμογή οικονομικής διαχείρισης (Budget Management Application),
αναπτυγμένη σε Java (JavaFX). Η εφαρμογή επιτρέπει την καταγραφή, ανάλυση και σύγκριση οικονομικών δεδομένων (έσοδα και έξοδα, κρατικές δαπάνες) με στόχο την καλύτερη κατανόηση και διαχείριση του ετήσιου προϋπολογισμού.

Το project έχει υλοποιηθεί στο πλαίσιο ακαδημαϊκού / εκπαιδευτικού σκοπού και ακολουθεί βασικές αρχές καθαρού κώδικα, διαχωρισμού λογικής και διεπαφής χρήστη, καθώς και χρήσης τοπικής βάσης δεδομένων.

ΔΙΑΧΕΙΡΙΣΗ ΧΡΗΣΤΩΝ

Η εφαρμογή υποστηρίζει:
- Εγγραφή νέου χρήστη (Sign Up)
- Είσοδο χρήστη (Login)
- Αποθήκευση δεδομένων ανά χρήστη στη βάση δεδομένων

-----------------------------------------------------------------------

ΛΕΙΤΟΥΡΓΙΕΣ ΕΦΑΡΜΟΓΗΣ

1. ΚΑΤΑΧΩΡΗΣΗ ΕΣΟΔΩΝ ΚΑΙ ΕΞΟΔΩΝ
- Καταγραφή ποσών ανά κατηγορία
- Επιλογή έτους
- Αυτόματος υπολογισμός συνόλων

2. ΑΝΑΛΥΣΗ ΟΙΚΟΝΟΜΙΚΩΝ
- Προβολή οικονομικών δεδομένων ανά έτος
- Συγκεντρωτική παρουσίαση στοιχείων
- Δυναμική ενημέρωση περιεχομένου

3. ΣΕΝΑΡΙΑ (SCENARIOS)
- Δημιουργία εναλλακτικών οικονομικών σεναρίων
- Προσομοίωση διαφορετικών επιλογών
- Αποθήκευση και σύγκριση σεναρίων

4. ΣΥΓΚΡΙΣΗ ΣΕΝΑΡΙΩΝ
- Αντιπαραβολή εσόδων και εξόδων
- Ανάδειξη διαφορών
- Υποστήριξη λήψης αποφάσεων

5. ΣΥΝΟΨΗ (SUMMARY)
- Συνολική εικόνα οικονομικής κατάστασης
- Καθαρό αποτέλεσμα (πλεόνασμα ή έλλειμμα)
- Ετήσια αναφορά προϋπολογισμού

-----------------------------------------------------------------------

ΒΑΣΗ ΔΕΔΟΜΕΝΩΝ

- Χρήση SQLite
- Τοπική αποθήκευση δεδομένων
- Δεν απαιτείται εξωτερικός server
- Αρχείο βάσης δεδομένων: database.db

-----------------------------------------------------------------------

ΔΙΕΠΑΦΗ ΧΡΗΣΤΗ (UI)

- Responsive σχεδίαση
